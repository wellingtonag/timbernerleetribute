Tim Berners Lee

The father of the World Wide Web

Tim Berners-Lee's time line

  • 1955: Tim Berners-Lee is born in London, United Kingdom.
  • 1973-1976: Berners-Lee studies Physics at Queen's College, University of Oxford.
  • 1976-1980: He works as a software engineer at various companies, including Plessey Telecommunications, D.G. Nash Limited, and Feranti Computer Systems.
  • 1980: Berners-Lee starts working as a software engineer at CERN (European Organization for Nuclear Research) in Geneva, Switzerland.
  • 1989: Berners-Lee proposes a hypertext-based information management system, which would become the foundation of the World Wide Web.
  • 1990: Berners-Lee develops HTML (Hypertext Markup Language), the markup language used to create web pages.
  • 1991: The first web server, called WorldWideWeb, is released. Berners-Lee also creates the first web browser, which allowed the viewing of web pages.
  • 1994: Berners-Lee founds the World Wide Web Consortium (W3C) to promote standards and the continuous evolution of the web.
  • 2009: Berners-Lee launches the Web Foundation, aiming to promote universal web access and ensure the web is used for the benefit of all humanity.
  • 2016: The Solid project is launched, led by Berners-Lee, with the goal of decentralizing the web and giving users control over their personal data.
  • 2021: Berners-Lee continues to work on various web-related initiatives, such as combating misinformation and promoting online privacy and security.

"The original idea of the web was that it should be a collaborative space where you can communicate through sharing information."

-- Tim Berners-Lee
